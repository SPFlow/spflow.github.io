DSL (Example Construction)
==========================

The :mod:`spflow.dsl` module provides a small, example-oriented expression layer for constructing
simple circuits with algebraic syntax. It is intended for documentation and showcase snippets;
production model construction is typically done directly via :mod:`spflow.modules`.

   :members: term, w, build, Term, ProductExpr, SumExpr, WeightedExpr
